LogicBuilder_CSharp_.net
Introduction

LogicBuilder_CSharp_.net is a menu-driven console application developed using C# and the .NET framework.
The project focuses on building strong programming logic by implementing commonly used computational problems in a structured and modular manner.

Each functionality is separated into individual classes, making the application easy to understand, maintain, and extend.

Project Objectives

To strengthen logical thinking using C#

To understand menu-driven program design

To apply object-oriented programming principles

To practice arrays, matrices, and number-based algorithms

To gain hands-on experience with .NET console applications

Application Functionalities

The application provides the following menu-based operations:

Arithmetic operations

Display size of data types

Swapping of two numbers

Number conversion

Arithmetic operations using switch case

Find largest, smallest and check odd/even

Swap of two-digit integer

Check divisibility by 5, 7, and 11

Leap year verification

Odd or even number check

Fibonacci series generation

Display array elements

Find largest and smallest element in an array

Calculate sum and average of numbers

Read a 3×3 matrix

Multiply two 3×3 matrices

Palindrome check

Exit the application

Technologies Used

Programming Language: C#

Framework: .NET

Application Type: Console Application
